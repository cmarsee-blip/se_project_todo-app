# To-Do App

A checklist utility app built with vanilla JavaScript, form validation, and a modular architecture.

🔗 **Live Demo:** [cmarsee-blip.github.io/se_project_todo-app](https://cmarsee-blip.github.io/se_project_todo-app/) &nbsp;|&nbsp; 🎥 **Demo Video:** [link](#) *(optional)*

---

## 📖 Overview

This project's goal was to build a functioning checklist/to-do application using vanilla JavaScript — practicing DOM manipulation, form handling, and organizing a codebase into reusable modules rather than one large script.

## 🛠️ What I Built & How

I built a to-do app where users can add, complete, and manage checklist items. The app uses form validators to handle user input and is organized into separate folders for blocks, components, pages, and utilities, keeping concerns separated instead of a single monolithic file.

**Key features:**
- Add / manage checklist items via a validated form
- Custom form validation (no library dependency)
- Modular code structure (blocks, components, pages, utilities)

**Built with:** JavaScript, HTML, CSS
